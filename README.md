# New Heart of UDM

## Backend API

Сервер **FastAPI** живёт в каталоге `backend/` и ведётся как **отдельный Git-репозиторий** (внутри `backend/.git`). См. [backend/README.md](backend/README.md): как запустить, как запушить на GitHub. В корневом `.gitignore` папка `backend/` игнорируется, чтобы не смешивать истории с Android-приложением.
